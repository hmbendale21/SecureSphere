# 🛡️ SecureSphere

A modern cybersecurity portal built using HTML, CSS, Bootstrap, and JavaScript.

## 🚀 Project Goal

SecureSphere is an interactive cybersecurity platform designed to help users improve their online security through useful tools, educational resources, and real-time cyber awareness.

## 👨‍💻 Tech Stack

- HTML5
- CSS3
- Bootstrap 5
- JavaScript
- Git & GitHub
